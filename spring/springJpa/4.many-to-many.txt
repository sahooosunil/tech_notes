## **Many-to-Many Relationship in Spring Data JPA**  

### **Understanding Many-to-Many**  
A **Many-to-Many** relationship means that multiple records in one entity are associated with multiple records in another entity.

ğŸ“Œ **Example Scenario:**  
- **Students** enroll in **multiple Courses**  
- **Courses** have **multiple Students**

## **1ï¸âƒ£ Define the Entities**  

### **Student Entity**
```java
@Entity
public class Student {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String name;

    @ManyToMany
    @JoinTable(
        name = "student_course",
        joinColumns = @JoinColumn(name = "student_id"),
        inverseJoinColumns = @JoinColumn(name = "course_id")
    )
    private Set<Course> courses = new HashSet<>();

    // Getters and Setters
}
```
âœ… **`@ManyToMany`** â†’ Many students can enroll in many courses.  
âœ… **`@JoinTable`** â†’ Creates a **junction table (`student_course`)** with **student_id** and **course_id** as foreign keys.  

---

### **Course Entity**
```java
@Entity
public class Course {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String title;

    @ManyToMany(mappedBy = "courses")
    private Set<Student> students = new HashSet<>();

    // Getters and Setters
}
```
âœ… **`mappedBy = "courses"`** â†’ This **avoids duplication** of the join table in both entities.

## **3ï¸âƒ£ Saving Data (Service Layer)**
```java
@Service
public class EnrollmentService {
    @Autowired
    private StudentRepository studentRepository;
    @Autowired
    private CourseRepository courseRepository;

    public void enrollStudents() {
        // Create Courses
        Course javaCourse = new Course();
        javaCourse.setTitle("Java Programming");

        Course dbCourse = new Course();
        dbCourse.setTitle("Database Management");

        courseRepository.saveAll(List.of(javaCourse, dbCourse));

        // Create Students
        Student student1 = new Student();
        student1.setName("Alice");

        Student student2 = new Student();
        student2.setName("Bob");

        // Assign Courses to Students
        student1.getCourses().add(javaCourse);
        student1.getCourses().add(dbCourse);

        student2.getCourses().add(javaCourse);

        // Save Students (Courses will also be saved)
        studentRepository.saveAll(List.of(student1, student2));
    }
}
```
ğŸ”¹ **Students and Courses are linked** through the `student_course` table.  
ğŸ”¹ **`student1` is enrolled in both Java and DB courses.**  
ğŸ”¹ **`student2` is enrolled only in Java.**
